# Board Games Recommendation System

**Goal:** This project builds a recommendation system using collaborative filtering to suggest items based on user preferences. The analysis focuses on handling sparse user-item interaction data and uncovering patterns in user behavior to support effective recommendations.

**Skills:** Data Cleaning, Exploratory Data Analysis (EDA), Matrix Factorization (SVD).
  
**Technology:** Python, Pandas, Numpy, Seaborn, Matplotlib, Surprise.
